# ISYMGANAK benchmarks

The problem instances (CNF) used to evaluate
[ISYMGANAK](https://github.com/ibrahimelk/isymganak),
a model counter that exploits symmetry with a 
multi-layer cache.

The same
[benchmark](https://github.com/VincentDerk/Paper-SymGANAK-benchmark)
instances was used for evaluating SYMGANAK, a model counter
introduced in AAAI2021 that exploits symmetry: Symmetric Component
Caching for Model Counting on Combinatorial Instances by
Timothy van Bremen, Vincent Derkinderen, Shubham Sharma,
Subhajit Roy, Kuldeep S. Meel.
